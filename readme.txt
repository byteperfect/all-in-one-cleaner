== Description ==

The All-In-One Cleaner plugin allows you to clean up your WordPress site. It provides various modules to clean different types of data.

The main difference between The All-In-One Cleaner plugin and other similar ones is that The All-In-One Cleaner plugin is able to delete a really huge amount of data without causing a timeout interrupt error.

In addition, the plugin has a very high speed of data processing. Tests show that on average the plugin correctly deletes 1000 records per minute.

== Installation ==

1. Log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
2. In the search field type “All-In-One Cleaner” then click “Search Plugins.” Once you’ve found us, click “Install Now”.
3. Activate the plugin.
4. Use the Tools->All-In-One Cleaner screen to configure the plugin.
